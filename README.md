# Memory-flash

## Overview
This repo is for the requirements for the software implementation of a game called Memory flash. The game challenges players to remember and repeat an increasing sequence of colors. Each round, the sequence grows, testing the player’s memory and attention. 

## the game
A player is presented with a sequence of colored buttons that light up in a pattern, generated by the game. The player must repeat the pattern by clicking the buttons in the exact order. Each round the sequence increases by one color; if the player reproduces the sequence correctly, they advance to the next round, but if they make a mistake, the game ends.
